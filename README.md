# exam_digest

A responsive web-based quiz application developed using HTML, CSS, JavaScript, and Firebase Firestore for conducting online examinations.

## Features

- Student authentication and validation
- Dynamic question loading
- Multiple-choice quiz interface
- Countdown timer with auto submission
- Real-time score calculation
- Firebase Firestore integration
- Prevention of multiple exam attempts
- Responsive user interface
- Android application support using WebView

## Technologies Used

- HTML
- CSS
- JavaScript
- Firebase Firestore
- Netlify
- Android WebView

## Project Workflow

1. Student login and verification
2. Question retrieval from Firebase
3. Quiz attempt with timer
4. Answer submission
5. Result storage in Firebase

## How to Run

1. Clone the repository
2. Open the project folder
3. Configure Firebase credentials
4. Run using Live Server or open `index.html`
